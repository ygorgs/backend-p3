Livro

A representação de um livro

GET /book

Retorna uma lista com todos os livros cadastrados.

POST /book

Cadastra um novo livro com os seguintes campos:

  • title: Título do livro;
  • authors: Lista com o(s) autor(es) do livro;
  • description: Descrição do livro;
  • price: Preço do livro;
  • imageUrl: Url da imagem da capa do livro.

GET /book/:id

Retorna o livro com o ID especificado.

PUT /book/:id

Substitui um livro existente que possui o ID especificado.

Comentários

A representação de um comentário dado a um determinado livro.

GET /book/:idBook/comment

Retorna uma lista de comentário associada ao livro com o ID especificado.

POST /book/:idBook/comment

Adiciona um novo comentário ao livro indicado através de seu ID. O comentário deve possuír os seguintes atributos:

  • bookid: ID do livro que esse comentário faz referência;
  • name: Nome da pessoa responsável pelo comentário;
  • text: Texto do comentário.
